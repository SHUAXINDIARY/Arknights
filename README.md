# 食用指南

## 编辑页

- 点击预览示例：可以以本人的数据查看结果页效果
- 点击生成：选择/填写至少一项数据后，可以生成自己的结果页

## 结果页

- 点击编辑：可以返回编辑页基于之前填写的数据继续编辑
- 点击返回：清空填写数据并返回编辑页
- 点击导出结果：自动对结果截图并提示下载（需要再浏览器打开才可用，app内打开目前只有飞书/微信测试可用，其他不行）
